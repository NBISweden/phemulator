# phemulator
Phemulator tool for visualising GWAS effect sizes in A_Johansson_PP

# Install and run:

```
docker run --rm -p 8787:8787 nbisweden/phemulator:v1.0.0
```

than go to browser and type `http://0.0.0.0:8787`
